Тест Рабина-Миллера для числа n = 45
Количество итераций: 12
==================================================
n-1 = 44 = 2^2 * 11
b = 2, m = 11

Итерация 1:
  Выбрано случайное a = 34
  j = 0, z = a^m mod n = 34^11 mod 45 = 4
  j = 1:
    z = z^2 mod n = 16
  После всех j, z ≠ n-1, число не простое
