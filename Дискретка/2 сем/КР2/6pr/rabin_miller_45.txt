Тест Рабина-Миллера для числа n = 45
Количество итераций: 5
==================================================
n-1 = 44 = 2^2 * 11
b = 2, m = 11

Итерация 1:
  Выбрано случайное a = 24
  j = 0, z = a^m mod n = 24^11 mod 45 = 9
  j = 1:
    z = z^2 mod n = 36
  После всех j, z ≠ n-1, число не простое
